Java toimii Helsingin yliopiston peruskurssien ohjelmointikielenä ja olenkin opetellut ohjelmoimaan Javalla. Java on myös kieli, jota olen käyttänyt ehkä eniten. Java web-ohjelmoinnissa olen käyttänyt apuna Spring Boot -sovellukehystä.

Uutissovellus

web Sovellus code Github

Olen tehnyt uutissovelluksen osana web-palvelinohjelmointi -kurssia. Uutissovelluksen ohjelmoinnissa apuna ovat toimineet Spring Boot, Thymeleaf ja H2-tietokantamoottori. Käytössä oli relaatiotietokanta, johon oliot tallennettiin Spring Data JPA:n avulla. Lisäksi kurssilla käytiin pintapuolisesti läpi Travisia ja Seleniumia testaamisen apuvälineinä. Ulkoasu on tehty Bootstrapin avulla.
