Olen oppinut Javascriptin yliopiston full stack -kurssin kautta. Kurssilla pääpaino oli "React-kirjaston avulla toteutettavissa single page -sovelluksissa, ja niitä tukevissa Node.js:llä toteutetuissa REST-rajapinnoissa." Tietokantana kurssilla toimi MongoDB, jota käytettiin Mongoose -kirjaston avulla. Kurssin jälkeen olen syventänyt oppimaani omatoimisesti pääasiallisesti Reactia ja NodeJS:ää käyttäen. Tämän esittelysivuston ohjelmoinnissa tutustuin kuitenkin pintapuolisesti myös Angular2-sovelluskehykseen.

Vaalikone

web Sovellus code Github (vanha ulkoasu)

Tämä on tällä hetkellä monimutkaisin tekemäni web-sovellus. Vaalikone on tehty Reactin ja NodeJS:n avulla. Selainpuolen tilanhallinnassa on käytetty Reduxia ja palvelinpuolella pyyntöjen käsittelyssä apuna on toiminut Express.

Lisätietoja

Sovelluksen ajatuksena on vertailla eduskuntaryhmien tekemien päätöksien osuvuutta vaalikoneen käyttäjän näkemyksiin. Lisäksi ajatuksena on, että käyttäjä voisi vertailla ryhmien ja edustajien käyttäytymistä ylen vaalikoneesta löytyneisiin vastauksiin.

Sovelluksessa onkin hyödynnetty ylen tarjomaa avointa dataa sen oman vaalikoneen vastauksista. Ylen data on parsittu ja käsitelty sovelluksen tarpeisiin sopivaksi. Käsitellystä datasta on sitten luotu rajapinta, jota sovellus käyttää. Eduskunnassa tehdyt päätökset avataan ja lisätään avattuina REST-rajapintaan lomakkeella.

Vaalikone on yhä työnalla ja olen saanut sen ulkoasun muokkaamiseen apua ystävältä. Nykyisen ulkoasun olen tehnyt tämän marvelapp -prototyypin pohjalta. Uuden ulkoasun koodi löytyy täältä.


Liiteri-animaatiot

web Animaatiot (ei responsiivisia, suunniteltu 892x1517 -tabletille )
code Github

Kaksi animaatiota, jotka on toteutettu React-paketteja hyödyntäen. Molemmissa on käytetty NodeJS:ää palvelinpuolella ja reduxia selainpuolen tilanhallinnassa.

Lisätietoja

Toisessa käytin react-dnd tarjoamia komponentteja ja loin simppelin "vedä ja pudota" -toiminnallisuuden. Jouduin myös käyttämään react-dnd-touch-backend -paketin tarjoamia komponentteja, sillä animaatiot tehtiin yrityksen mainostablettia varten.

Toisessa animaatiossa käytin GreenSock Animation Platform:n gsap-pakettia ja react-lottie -pakettia, joiden avulla loin simppelin pyöräilijä animaation. Paketteja ei ole käytetty aivan optimaalisella tavalla ja käytännössä hyödynsinkin lähinnä setTimeout() metodia ja Reactin routeria. Routeriakaan ei tosin ole käytetty aivan parhaalla mahdollisella tavalla.

Animaatiot ovat itsenäisen työn tulos ja apu olisi varmasti helpottanut työtä. Opin kuitenkin paljon ja seuraavalla kerralla ohjelmointi jälkikin on varmasti siistimpää. Ohjelmoinnin lisäksi opin -projektissa myös paljon yleistä startup-yrityksen ja monikansallisten tiimien toiminnasta. Demola -projektista lisää täällä.
